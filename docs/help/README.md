# 开发者帮助

本目录面向开发者，提供核心模块的功能概览与使用入口，便于快速定位配置、日志、国际化与远程升级相关能力。

## 模块总览
- [配置模块](./config.md)：系统配置与应用配置的职责划分、注入方式与运行时读写流程。
- [日志模块](./logger.md)：基于 slog 的日志能力、输出策略与 Wails 适配方式。
- [国际化模块](./i18n.md)：语言配置、前端词条定义与设置接口。
- [远程升级模块](./updater.md)：升级检查协议、下载与安装流程、系统配置入口。

## 适用范围
- Core 能力：配置、日志、升级等核心模块的开发与集成
- Backend 适配：Wails API 桥接与运行时装配
- Frontend：设置与帮助页的展示与调用
