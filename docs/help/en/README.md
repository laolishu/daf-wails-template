# Developer Help

This folder targets developers and provides quick entry points for core modules so you can locate configuration, logging, i18n, and remote update guidance quickly.

## Module Overview
- [Configuration](./config.md): system vs app config responsibilities, injection, and runtime read/write flow.
- [Logging](./logger.md): slog-based logging, outputs, and Wails adapter integration.
- [Internationalization](./i18n.md): language settings, frontend message catalog, and backend APIs.
- [Remote Update](./updater.md): update check protocol, download/install flow, and system entry points.

## Scope
- Core capabilities: config, logging, updater modules
- Backend adapters: Wails API bridge and runtime wiring
- Frontend: settings/help pages and calls
