# 国际化模块

## 功能定位
国际化能力由应用配置与前端词条表共同驱动：
- 应用配置中保存语言偏好（`i18n.language`）。
- 前端词条定义在 `frontend/src/i18n.ts`，当前内置 `zh-CN` 与 `en-US`。
- 后端提供语言读取与更新接口，供前端设置页调用。

## 主要使用方式
- 使用 `config.Manager` 读取或更新语言配置。
- 通过后端 `App.GetLanguage`/`App.SetLanguage` 暴露给前端。
- 前端使用 `normalizeLanguage` 与 `messages` 完成文案切换。

## 关键配置或接口
- 配置项：`i18n.language`（默认 `zh-CN`，支持 `en-US`）
- 后端接口：`App.GetLanguage()`、`App.SetLanguage(language)`
- 前端入口：`frontend/src/i18n.ts`
