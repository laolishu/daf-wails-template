# 变更：新增开发者帮助文档

## 为什么
当前文档分散在不同主题中，缺少面向开发者的一站式入口。新增开发者帮助文档可提供模块总览与清晰的使用指引，降低上手成本。

## 变更内容
- 在 docs/help 下新增开发者帮助文档目录
- 提供总纲页，概述配置、日志、国际化、远程升级模块并链接到详细页
- 提供各模块的详细说明页，覆盖功能、使用方式与相关链接

## 影响
- 受影响规范：developer-help-docs（新增）
- 受影响文档：docs/help/README.md、docs/help/config.md、docs/help/logger.md、docs/help/i18n.md、docs/help/updater.md
