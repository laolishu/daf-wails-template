# Design Doc：顶部标题栏（可拖动）

## 目标
在保持现有 Sidebar + 主内容区结构的前提下，为桌面应用增加统一的顶部标题栏，用于品牌展示与窗口拖动。

## 设计要点
1. **视觉一致性**
   - 标题栏背景色与 Sidebar 相同，形成整体统一的框架感。
   - 标题栏内包含 Logo 与 title，title 取自 `window.title`。

2. **尺寸约束**
   - 高度遵循 56–64px 区间，避免过高或过窄。

3. **拖动区域**
   - 标题栏需要标记为可拖动区域，确保用户可拖动窗体。
   - 对内部交互元素（如按钮）需设置为不可拖动区域以避免误操作（如有）。

## 影响范围
- 前端布局：新增 AppShell.Header 与标题栏样式
- Wails 窗口：配置拖动区域（CSS 或运行时标记）

## 兼容性
- 不破坏现有 Sidebar 与主内容结构，仅增加顶部区域。
