# frontend-shell 规范增量：顶部标题栏

## 新增需求

### 需求：顶部标题栏
应用必须提供顶部标题栏，用于品牌展示与窗口拖动。

#### 场景：渲染顶部标题栏
- **当** 应用渲染主界面
- **那么** 必须显示顶部标题栏
- **并且** 标题栏背景色必须与 Sidebar 相同
- **并且** 标题栏包含 Logo 与 title，title 必须来自 `window.title`
- **并且** 标题栏高度必须在 56–64px 区间

---

### 需求：标题栏拖动区域
标题栏必须支持窗体拖动。

#### 场景：用户拖动窗体
- **当** 用户在标题栏区域按住并拖动
- **那么** 必须能够拖动窗体
- **并且** 标题栏内的可交互元素（如按钮）不得阻断拖动行为
