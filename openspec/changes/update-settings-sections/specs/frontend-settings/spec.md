## 新增需求

### 需求：Settings 栏目划分
Settings 页面必须划分为“通用设置”和“系统信息”两个栏目。

#### 场景：用户浏览 Settings
- **当** 用户查看 Settings 页面
- **那么** 必须看到“通用设置”和“系统信息”两个栏目分区

## 修改需求

### 需求：语言选项（首项）
Settings 页面必须在“通用设置”栏目中提供语言选项并置于第一项，仅支持中文（zh-CN）与英文（en-US）切换，且语言选项必须单行显示与操作。

#### 场景：用户查看 Settings 语言选项
- **当** Settings 页面渲染
- **那么** 语言选项必须位于通用设置的第一项、仅包含 zh-CN/en-US，且单行展示与操作
