# Design Doc：左侧导航栏密度与对齐

## 目标
提升 Sidebar 的空间效率与专业感，形成更紧凑但不拥挤的桌面导航体验。

## 设计要点
1. **宽度与对齐**
   - Sidebar 宽度收敛到 200–220px，默认 208px。
   - 纵向布局对齐，减少横向空白。

2. **菜单项结构**
   - 高度 40px，保证清晰点击区域。
   - 图标尺寸 18–20px，左边距 16px，图标与文字间距 10–12px。

3. **交互与选中态**
   - 选中态使用浅色圆角块（8px），内部内边距左右 8–10px、上下 6px。
   - Hover 轻微提亮，不改变尺寸；Active 仅颜色加深，避免高对比描边。

4. **风格定位**
   - 保持 CCleaner / 1Password / Linear 的克制与可信风格，紧凑但不拥挤。

## 影响范围
- 前端布局与样式：Sidebar 宽度与菜单项间距
- 规范：frontend-guidelines、frontend-shell、frontend-ui
