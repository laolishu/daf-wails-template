# 变更：优化左侧导航栏密度与对齐

## 为什么
当前 Sidebar 在宽度与间距上偏松，横向空白较多，影响信息密度与专业感。需要收敛导航栏宽度与菜单项间距，保证更紧凑但不拥挤的桌面体验。

## 变更内容
- Sidebar 宽度调整为 200–220px（默认 208px）
- 菜单项高度统一为 40px
- 图标尺寸 18–20px，图标左边距 16px，图标与文字间距 10–12px
- 选中态为浅色圆角块（8px），内边距左右 8–10px、上下 6px
- Hover 仅轻微提亮，不改变布局尺寸；Active 颜色加深，不增加边框厚度

## 影响
- 受影响规范：frontend-guidelines、frontend-shell、frontend-ui
- 受影响代码（后续实施阶段）：frontend/src/App.tsx
