# frontend-guidelines 规范增量：Sidebar 密度与对齐

## 修改需求

### 需求：Sidebar 尺寸与菜单元素
Sidebar 必须采用固定宽度与尺寸范围，确保左侧导航的一致性。

#### 场景：渲染侧边栏导航
- **当** 渲染侧边栏
- **那么** Sidebar 宽度必须为 200–220px，默认 208px
- **并且** 菜单项高度必须为 40px
- **并且** 菜单图标尺寸必须在 18–20px 范围内
- **并且** 菜单图标左边距必须为 16px
- **并且** 菜单图标与文字间距必须为 10–12px

---

### 需求：选中态与交互状态
Sidebar 的选中态与交互状态必须克制且不改变布局尺寸。

#### 场景：菜单项交互
- **当** 菜单项处于 Hover 状态
- **那么** 背景仅轻微提亮，不改变布局尺寸
- **当** 菜单项处于 Active 状态
- **那么** 颜色加深且不增加边框厚度
- **并且** 禁止强描边或高对比边框
